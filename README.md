                                                      Cat Rescue
Cat Rescue is a website with a format that is geared towards animal shelters. It uses HTML, CSS, Flexbox and JavaScript to showcase information. Cat Rescue uses a flexbox layout to ensure a responsive experience.
It features an About Segment, Program Information, Aside Area and a Photo Gallery.
